    php序列化

    主要是两个函数

    string serialize ( mixed $value )

    $value: 要序列化的对象或数组。

    返回值是序列化后的字符串

    mixed unserialize ( string $str )

    $str: 序列化后的字符串。

    返回的是转换之后的值，可为 integer、float、string、array 或 object。

    如果传递的字符串不可解序列化，则返回 FALSE，并产生一个 E_NOTICE。
